# character-count 

Count characters in a string. Build with __nan__.

## Install 

```sh

# With npm 
npm i character-count

```

## How to use 

```js 

const characterCount = require('character-count');

characterCount('pizza', 'z') // ~> 2
characterCount('hello world', 'l') // ~> 3
characterCount('foo(bar(baz)', '(') // ~> 2

```
